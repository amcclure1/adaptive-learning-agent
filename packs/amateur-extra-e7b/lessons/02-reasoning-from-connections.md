# Reason from transistor connections

A resistor's role follows from its endpoints, not its page position. A pair of resistors spanning a supply and ground can establish a voltage at their shared node. A resistor in a transistor's current path can develop a changing voltage that opposes operating-point drift. A capacitor connected across that path can provide a different route for changing signals while leaving the direct-current path intact.

Amplifier configuration is named for the transistor terminal common to the signal input and output paths. Trace where the input signal is coupled into the device, where the output signal is coupled out, and which device terminal returns toward the common reference. This method distinguishes the configuration from other arrangements without relying on the drawing's orientation.
