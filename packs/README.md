# Subject Pack Workspaces

This directory reserves source-controlled pack templates and pilot-pack workspaces. Pack files are portable content, not operational learner state. Do not place credentials, databases, attempt histories, Hermes configuration, or proprietary examination questions here.

`fixture-basics/` is a functional synthetic version-0.1 pack used only by core tests and examples. The AWS and Amateur Extra directories contain no functional learning content. The accepted initial format is documented in `docs/mvp-vertical-slice.md`.
