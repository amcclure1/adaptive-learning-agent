# Question Independent-Verification Protocol

Status: future required protocol

Independently verify every scenario fact, constraint, option, key, rationale, explanation, citation, objective/blueprint mapping, and originality boundary. Confirm that the keyed option or set satisfies every explicit requirement, each distractor fails for a documented current reason, no hidden assumption is required, selection counts agree, and no protected or suspicious expression is reproduced.

Reopen authoritative sources; do not trust claim names alone. Findings bind exact question and dependency digests. The verifier cannot grant question-content approval, rewrite the question, or infer uniqueness approval.
