# Adaptive Learning Agent 0.2.0-alpha.1 — Sourced Content Pilot

Release date: 2026-07-19
Status: pre-release
Git tag: `v0.2.0-alpha.1`
Python distribution version: `0.2.0a1`

## Purpose

This release proves that Adaptive Learning Agent can deliver a real, authoritative, sourced learning pack through its runtime-independent Python core and Hermes conversational runtime. Deterministic Python tools remain responsible for validation, scoring, and learner state; SQLite remains authoritative operational storage; Hermes provides presentation and workflow.

## Features

- Explicit sourced pack format 0.2 with strict closed-record validation and deterministic canonical digests.
- Format-0.1 backward compatibility, including an unchanged golden digest.
- Multiple Markdown lessons presented in declared order.
- Official question identity and explicit `official_pool` or `generated` origin semantics.
- Structured source and citation records with dates, revisions, locators, URLs, and conditional retained-snapshot digests.
- Assessment-pool identity, effective dates, declared errata, and withdrawn-question metadata.
- Component-level `public_domain`, `licensed`, and `reference_only` rights records.
- Required digest-covered human approval before format-0.2 validation or installation.
- Answer-safe pre-submission delivery and deterministic post-answer provenance.
- Approved NCVEC Amateur Extra E1A pilot containing E1A01–E1A11, two objectives, and two original lessons.
- Real Hermes v0.18.2 delivery, source display, independent restart reconstruction, challenge quarantine, and immutable-attempt verification.

The ten-operation runtime-neutral tool contract remains version 0.1. Format 0.2 adds capabilities and provenance without changing SQLite schema 1 or deterministic learning behavior.

## Content provenance

The official baseline is the **2024–2028 Extra Class FCC Element 4 Question Pool and Syllabus**, effective July 1, 2024 through June 30, 2028, maintained by the National Conference of Volunteer Examiner Coordinators Question Pool Committee. The consolidated source incorporates the fourth errata dated February 4, 2026. Immediately before release, the [official NCVEC release page](https://www.ncvec.org/index.php/2024-2028-extra-class-question-pool-release) still identified that fourth errata as current and showed no newer change affecting E1A.

NCVEC states that the 2024–2028 Element 4 Extra Class Question Pool is released into the public domain. The pack limits that rights scope to official IDs, wording, ordered choices, and answer keys. Current 47 CFR Part 97 material from FCC via eCFR is the regulatory baseline; retained source metadata records the point-in-time review date and digest. Original lessons and project-authored explanations use the project's approved original-prose license, CC-BY-4.0. External sources are reference-only metadata and links.

E1A06 preserves the printed NCVEC locator `[97.303(h)(1)]` exactly. Its project-authored explanation separately cites current 47 CFR § 97.303(h)(3), the operative paragraph for the CW center-frequency requirement. Project-authored lessons and explanations are not official NCVEC commentary.

## Limitations

- Only question group E1A is included; this is not a complete Amateur Extra course or general examination-preparation claim.
- No exam simulation, readiness prediction, mastery model, scheduling, or adaptive sequencing.
- No conversational subject builder or generated-question workflow.
- Content rights have not received formal legal review; the recorded policy is a project decision, not legal advice.
- Only the Hermes v0.18.2 Windows CLI with the isolated `adaptive-learning-dev` profile is verified.
- Hermes Desktop, gateway, Linux, macOS, other Hermes versions, and other runtimes remain unverified.
- One local learner and at most one active session; no hosted multi-user or concurrent-process guarantee.
- No application backup/restore workflow, encryption at rest, or hostile same-account-process isolation.

## Installation and use

### Install or use the tagged project

Python 3.12 or newer and Git are required. From PowerShell:

```powershell
git clone --branch v0.2.0-alpha.1 https://github.com/amcclure1/adaptive-learning-agent.git
Set-Location adaptive-learning-agent
python -m venv .venv
& .\.venv\Scripts\Activate.ps1
python -m pip install .
python -m unittest discover -s tests -v
```

The core has no third-party runtime dependencies. Building from the source checkout uses setuptools as declared in `pyproject.toml`.

### Validate and install the E1A pack through the core

The following tested public application boundary writes only to the repository's ignored `user-data` directory:

```powershell
@'
from pathlib import Path
from adaptive_learning.application_service import ApplicationService

service = ApplicationService(Path("user-data"))
print(service.system_health())
print(service.pack_validate(Path("packs/amateur-extra-e1a")))
print(service.pack_install(Path("packs/amateur-extra-e1a")))
'@ | python -
```

### Launch the isolated Hermes development profile

The verified path assumes pinned Hermes v0.18.2 and an existing isolated `adaptive-learning-dev` profile. From the trusted repository root:

```powershell
$env:HERMES_ENABLE_PROJECT_PLUGINS='1'
& "$env:USERPROFILE\.local\bin\hermes.exe" -p adaptive-learning-dev --skills adaptive-learning:adaptive-learning
Remove-Item Env:HERMES_ENABLE_PROJECT_PLUGINS
```

Project plugins execute repository code. Keep the discovery gate process-local and do not run from an untrusted checkout.

In Hermes, request these steps conversationally:

1. “Validate and install the pack at `packs/amateur-extra-e1a`.”
2. “Initialize my local learner and begin studying `us-amateur-extra-e1a` version `0.2.0`.”
3. Answer the presented option and provide confidence from 1 through 5.
4. After scoring, ask “show sources.”
5. Exit Hermes, relaunch with the same command, and ask to resume the active study session from persisted learner state.

Hermes conversation and memory are not learner state. Resume behavior must come from the deterministic health, learner, and study tools backed by SQLite.
